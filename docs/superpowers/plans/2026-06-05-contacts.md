# Contacts Feature Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a polymorphic `Contact` model (first name, last name, email) linked to customers via a `contactables` pivot, with a standalone index + show page and a contacts section on the customer show page.

**Architecture:** Morph-pivot pattern (`contactables` table) identical to `imageables`/`documentables`. `Contact` model with `morphedByMany(Customer)`, `Customer` model gets `morphToMany(Contact)`. `ContactPolicy` auto-discovered by Laravel convention. `ContactController` follows the `ProductController` index+show pattern; requests use `$this->user()->can(...)` to delegate to the policy.

**Tech Stack:** Laravel 12, Inertia, Vue 3, `@heroicons/vue/24/outline`, `@inertiajs/vue3`

## Task 4: ContactController + Route

**Files:**
- Create: `app/Http/Controllers/ContactController.php`
- Modify: `routes/web.php`

- [ ] **Step 1: Create ContactController**

`app/Http/Controllers/ContactController.php`:
```php
<?php

namespace App\Http\Controllers;

use App\Models\Contact;
use App\Models\Customer;
use App\Http\Requests\ContactReadRequest;
use App\Http\Requests\ContactStoreRequest;
use App\Http\Requests\ContactUpdateRequest;
use App\Http\Requests\ContactDestroyRequest;

class ContactController extends Controller
{
    public function index(ContactReadRequest $request)
    {
        $search = trim((string) $request->input('search', ''));

        $contacts = Contact::with('customers')
            ->when($search !== '', fn($q) =>
                $q->where(fn($inner) =>
                    $inner->where('first_name', 'like', "%{$search}%")
                          ->orWhere('last_name', 'like', "%{$search}%")
                          ->orWhere('email', 'like', "%{$search}%")
                )
            )
            ->orderBy('last_name')
            ->orderBy('first_name')
            ->paginate(25)
            ->appends(['search' => $search]);

        $all_customers = Customer::orderBy('name')->get(['id', 'name']);

        return inertia('Contacts/IndexPage', [
            'contacts'     => $contacts,
            'allCustomers' => $all_customers,
            'search'       => $search,
        ]);
    }

    public function show(ContactReadRequest $request, Contact $contact)
    {
        $contact->load('customers');

        return inertia('Contacts/ShowPage', [
            'contact' => $contact,
        ]);
    }

    public function store(ContactStoreRequest $request)
    {
        $data = $request->validated();
        $customer_id = $data['customer_id'];
        unset($data['customer_id']);

        $contact = Contact::create($data);
        Customer::findOrFail($customer_id)->contacts()->attach($contact->id);

        return redirect()->back()->with('success', 'Contact aangemaakt.');
    }

    public function update(ContactUpdateRequest $request, Contact $contact)
    {
        $contact->update($request->validated());

        return redirect()->back()->with('success', 'Contact bijgewerkt.');
    }

    public function destroy(ContactDestroyRequest $request, Contact $contact)
    {
        $contact->delete();

        return redirect()->back()->with('success', 'Contact verwijderd.');
    }
}
```
